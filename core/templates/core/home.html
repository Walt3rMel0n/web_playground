Bienvenido a la web patio de juegos.


En esta web consta de 6 apps listadas a continuación.

    App core

    La app principal que sostiene el proyecto.

    App registration

    Esta app permite la creación de nuevos usuarios, y así poder aprovechar las funcionalidades del resto de la web. Se puede registrar con un nombre de usuario, una contraseña y un correo electrónico, el correo puede ser real o ficticio. Pero para la función de restablecer contraseña necesita un correo real. Podrá iniciar sesión, y cerrar sesión cuando lo desee. Otras funciones de la app son: cambiar de contraseña si así lo desea, cambiar el email y con la ayuda de un correo real podrá restablecer la contraseña si la ha olvidado.

    Si desea aprovechar las funciones de la web, pero sin crear un usuario, puede usar el usuario de ejemplo, iniciando sesión con el usuario: walterMelon, y la contraseña walter1234. En este usuario también encontrará unas con conversaciones ficticias de ejemplo.

    App profiles

    Con esta app puede personalizar su perfil, agregar o cambiar una foto de perfil si así lo desea, agregar o modificar una biografía que los demás usuarios podrán leer, y un enlace a una red personal. Esta app también permite ver la lista de todos los usuarios registrados, así como seleccionar cualquier perfil y ver su biografía y dar clic al enlace de su red personal y ser redireccionado.

    App messenger

    Messenger es una mensajería en tiempo real entre usuarios registrados.

    App pages

    Es una app que utiliza un crud para administrar noticias. Cualquier usuario puede crear, modificar o eliminar una noticia.

    App blog

    Consta de un blog del que se puede filtrar a partir de temas. Cualquier usuario registrado puede crear un blog, añadirle una imagen si lo desea y elegir uno o más temas para el filtro.
